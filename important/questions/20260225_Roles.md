# Roles

## SuperAdmin

* Can create, update, delete projects
* Can update roles and permissions
* Can do all other functions of Admin Project


## Admin Project

* Can create/update/delete clients
* Can create/update/delete appointments
* Can create/update/delete invoices
* Can create/update/delete groups
* Can create/update/delete documents

## Client

* Can read/ask for appointments
* Can read invoices
* Can read group permit
* Can read/(update) own documents
* Can create own profile and account
